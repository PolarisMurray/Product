# 🍎 Mac 用户启动指南

## 方法一：双击启动（推荐）✨

**直接双击 `启动应用.command` 文件**

这个文件在 macOS 上可以双击直接运行，会自动打开终端并执行。

如果双击后提示"无法打开"，请按以下步骤操作：

1. 右键点击 `启动应用.command` 文件
2. 选择"打开方式" → "终端"
3. 如果提示"无法验证开发者"，点击"打开"即可

## 方法二：终端运行

如果双击不行，可以在终端中运行：

```bash
cd /Users/alex/github/Product
./启动应用.command
```

或者：

```bash
cd /Users/alex/github/Product
bash 启动应用.sh
```

## 方法三：使用启动页面

1. 双击打开 `启动页面.html` 文件
2. 查看服务器状态
3. 点击"打开应用"按钮

## ⚠️ 常见问题

### 问题1：双击后没有反应

**解决方案**：
1. 右键点击文件 → "显示简介"
2. 在"打开方式"中选择"终端"
3. 勾选"更改所有..."

### 问题2：提示"权限被拒绝"

**解决方案**：
在终端运行：
```bash
chmod +x 启动应用.command
chmod +x 启动应用.sh
```

### 问题3：提示"无法验证开发者"

**解决方案**：
1. 系统设置 → 隐私与安全性
2. 找到被阻止的应用
3. 点击"仍要打开"

## 📝 手动启动（如果自动启动失败）

### 终端1 - 启动后端
```bash
cd backend
pip3 install -r requirements.txt  # 首次运行
python3 -m uvicorn main:app --reload
```

### 终端2 - 启动前端
```bash
npm install  # 首次运行
npm run dev
```

然后打开浏览器访问：http://localhost:5173
