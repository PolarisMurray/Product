# 🚀 BioReport Copilot - 一键启动指南

## ✨ 最简单的方式：一键启动

### Windows 用户
**直接双击 `启动应用.bat` 文件**，脚本会自动：
1. 检查并安装依赖
2. 启动后端服务器
3. 启动前端服务器
4. 自动打开浏览器

### Mac 用户
**直接双击 `启动应用.command` 文件**（推荐方式）

如果双击后提示无法打开：
1. 右键点击文件 → "打开方式" → "终端"
2. 或者在终端运行：`./启动应用.command`

脚本会自动：
1. 检查并安装依赖
2. 启动后端服务器
3. 启动前端服务器
4. 自动打开浏览器

### Linux 用户
在终端运行：
```bash
./启动应用.sh
```

## 🌐 启动页面

也可以先打开 `启动页面.html` 查看服务器状态和快速链接。

## 📋 手动启动方式（如果一键启动不工作）

### 前提条件

1. **安装 Node.js**
   - 访问：https://nodejs.org/
   - 下载 LTS 版本（推荐）
   - 安装后重启终端

2. **安装 Python 3.8+**
   - 访问：https://www.python.org/
   - 下载并安装

### 启动步骤

#### 终端 1 - 启动后端
```bash
cd backend
pip install -r requirements.txt  # 首次运行需要
python -m uvicorn main:app --reload
```

#### 终端 2 - 启动前端
```bash
npm install  # 首次运行需要
npm run dev
```

#### 打开浏览器
访问：**http://localhost:5173**

## 🔗 重要链接

- **前端应用**: http://localhost:5173
- **后端API**: http://localhost:8000
- **API文档**: http://localhost:8000/docs
- **健康检查**: http://localhost:8000/health

## ❓ 常见问题

### 为什么不能直接双击 index.html？

这是一个 **Vite + React** 项目，需要开发服务器来：
- 编译 JSX 和 TypeScript
- 处理 ES6 模块
- 编译 Tailwind CSS
- 提供热重载功能

### 端口被占用怎么办？

**后端端口（8000）**：
```bash
python -m uvicorn main:app --reload --port 8001
```

**前端端口（5173）**：
Vite 会自动选择其他可用端口

### 依赖安装失败？

1. 确保网络连接正常
2. 尝试使用国内镜像：
   ```bash
   # npm 镜像
   npm config set registry https://registry.npmmirror.com
   
   # pip 镜像
   pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
   ```

## 💡 提示

- 开发服务器支持**热重载**，修改代码后会自动刷新
- 不要关闭运行服务器的终端窗口
- 按 `Ctrl + C` 可以停止服务器
- 使用一键启动脚本最方便！
